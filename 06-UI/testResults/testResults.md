# 06-UI/testResults

## Purpose

Contains test results and validation reports for the UI layer: E2E tests, component tests, UI loop iteration results, and user experience validation.

## What belongs here

- E2E test output (if applicable)
- UI loop iteration results
- Component test reports
- User-facing validation reports

## What does NOT belong here

- Backend test results (belongs to `05-Supabase/testResults/` or `04-Normalizer/testResults/`)
- Queue smoke tests (belongs to `03-Queue/testResults/`)

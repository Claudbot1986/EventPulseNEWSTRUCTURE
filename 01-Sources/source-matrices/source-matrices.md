# Source Matrices

## Purpose

Evaluation criteria and comparison data for sources. Provides structured assessment framework for source adoption decisions.

## What Belongs

- Path comparison data
- Method effectiveness scores
- Source reliability scores
- Priority ranking data

## What Does Not Belong

- Raw test results (go to `diagnostics/`)
- Candidate records (go to `candidate-lists/`)

## Path Priority (per goals.md)

1. JSON-LD (highest)
2. Network
3. HTML
4. Render
5. Manual (lowest)

## Network Path Criteria (per network-path-strategy.md)

Network Path is preferred when:
- Cleaner output
- More complete data
- More stable over time

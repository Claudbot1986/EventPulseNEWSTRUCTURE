# Archive

## Purpose

Rejected or deprecated sources. Long-term storage of sources that failed evaluation or were superseded.

## What Belongs

- Discarded sources with documented reasons
- Failed phase 1 candidates
- Deprecated adapters
- Superseded source implementations

## What Does Not Belong

- Active sources (go to `active/`)
- Candidates under evaluation (go to `candidate-lists/`)

## Common Rejection Reasons

- `robots.txt` blocked
- No JSON-LD and no alternative path available
- API key required but unavailable
- Failed all three testing phases
- Deprecated in favor of a newer adapter
